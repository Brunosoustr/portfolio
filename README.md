# Data Portfolio - [Seu Nome]

Bem-vindo ao meu portfólio de análise de dados. Este site foi desenvolvido para demonstrar minhas habilidades técnicas em Python, SQL, Power BI e visualização de dados.

## 🚀 Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Animações:** [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/) com ScrollTrigger.
- **Data Science (Projetos):** Python (Pandas, Matplotlib, Seaborn), SQL, Jupyter Notebooks.

## 🛠️ Destaques do Projeto
- **Scroll Horizontal:** Seção "Sobre Mim" interativa com efeito de linha do tempo.
- **Dark Mode Nativo:** Design moderno focado em legibilidade e estética tech.
- **Responsividade:** Totalmente adaptado para dispositivos móveis.

## 📁 Estrutura de Pastas
- `/Projetos`: Contém as páginas detalhadas e artefatos (Notebooks, Scripts) de cada análise.
- `/assets`: Imagens, ícones e SVGs.
- `/css` & `/js`: Estilos e lógica de animação.
